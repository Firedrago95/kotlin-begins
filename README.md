# 코틀린 단기 속성 학습

- 이 문서는 자바 스프링 숙련자가 코틀린 스프링 과제전형 수행을 위해 속성 코틀린/스프링 학습 훈련 기록입니다.
- 주요 학습 키워드를 간략히 기록하고, 상세 내용은 블로그 포스팅 링크로 연결합니다.

---

## 📝 [1일차] 코틀린 스프링 프로젝트 초기 세팅
*   **`kotlin-spring` 플러그인:** 코틀린의 `final` 클래스를 스프링 CGLIB 프록시 생성을 위해 `open` 처리.
*   **`kotlin-jpa` 플러그인:** JPA 엔티티에 필요한 기본 생성자(No-arg)를 컴파일 타임에 자동 생성.
*   **`jackson-module-kotlin`:** 기본 생성자/Setter가 없는 코틀린 불변 객체의 JSON 역직렬화 지원.
*   **블로그 링크:** [여기에 링크 추가]()
